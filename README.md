# Health-Lifestyle-Coach
An end-to-end Health &amp; Lifestyle Recommender System built with Python, utilizing Machine Learning and a Streamlit UI to provide personalized recommendations and fitness insights for gym beginners.
## üåü Key Features
* **Personalized Scoring:** Calculates a "Lifestyle Score" (0-10) based on BMI, hydration, meal frequency, and activity.
* **Diet-Focused Insights:** Provides targeted nutritional advice (e.g., protein targets, calorie adjustments) based on the user's specific BMI category.
* **Interactive UI:** A clean, responsive dashboard built with Streamlit for real-time health profile visualization.
* **Synthetic Data Generation:** Includes a custom-built script to generate 400+ rows of realistic healthcare data using NumPy and Pandas.
